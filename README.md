# 云盘资源搜索

一个干净整洁、动画流畅的云盘资源搜索工具，支持夸克网盘、百度网盘、阿里云盘、迅雷网盘等多平台资源检索。

## 功能特性

- 多平台搜索：支持夸克、百度、阿里、迅雷等主流网盘
- 实时筛选：按网盘类型快速过滤结果
- 排序功能：支持最新优先、最早优先排序
- 深色模式：支持明暗主题切换，自动跟随系统
- 响应式设计：完美适配桌面端和移动端
- 流畅动画：精心设计的过渡动画效果
- 热门搜索：一键搜索热门关键词
- 加载更多：分页加载，浏览更流畅

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- 无框架依赖，纯静态页面
- CSS 变量实现主题切换
- CSS 动画与过渡效果
- Fetch API 异步请求

## API 来源

数据来源于 [泽索搜 Zreso](https://zreso.cn) 开放 API 接口

- 接口地址：`https://zreso.cn/api/search`
- 请求方式：GET
- 参数：
  - `q`：搜索关键词
  - `sort`：排序方式（latest / oldest）
